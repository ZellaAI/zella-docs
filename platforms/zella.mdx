## Supported Features

### Chat Completion

<CodeGroup>
```bash Curl
curl --request POST \
--url https://gateway.zella.ai/v1/chat/completions \
--header 'content-type: application/json, authentication: Bearer **********' \
--data '{
"user": "user_jskjf93o101",
"model": {
    "platform": "zella",
    "name": "meta-llama/Llama-2-7b-chat-hf"
},
"query": {
    "messages": [
        {
        "role": "system",
        "content": "You are a helpful assistant to help design api structure"
        },
        {
        "role": "user",
        "content": "Hi There!"
        }
    ]
},
"response": {
    "format": "json_object",
    "stream": true
    }
}'
````

```python Python
from zella import ZellaAI

zella_ai = ZellaAI(api_key="*****************")

user = "2312re3r33e33"
model = {
    "platform": "zella",
    "name": "meta-llama/Llama-2-7b-chat-hf"
}
query = {
    "messages": [ {
        "role": "system",
        "content": "You are a helpful assistant to help design api structure"
    },
    {
        "role": "user",
        "content": "Hi There!"
    }]
}
response = {
    "format": "json_object"
}
response = zella_ai.chat.completions.create(user=user, model=model, query=query, response=response)
```
