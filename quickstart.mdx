## Setup your access token

To get started, create an account on [Zella AI](https://zella.ai/) and get your Zella access token.

## Choose a model

Start your journey with Zella by selecting from [ wide range of supported models and platforms](/models) available to use in Zella.
Choose a standard model for simplicity, or personalize it with options like fallbacks and retries using the Zella dashboard.
[Discover the possibilities here.](/models)

## Make your first chat completion request.

<CodeGroup>
```bash Curl
curl --request POST \
--url https://gateway.zella.ai/v1/chat/completions \
--header 'content-type: application/json' \
--data '{
"user": "user_jskjf93o101",
"model": {
    "platform": "openai",
    "name": "gpt-3.5-turbo"
},
"query": {
    "messages": [
        {
        "role": "system",
        "content": "You are a helpful assistant to help design api structure"
        },
        {
        "role": "user",
        "content": "Hi There!"
        }
    ]
},
"response": {
    "format": "json_object",
    "stream": true
    }
}'
````

```python Python
from zella import ZellaAI

zella_ai = ZellaAI(api_key="*****************")

user = "2312re3r33e33"
model = {
    "platform": "openai",
    "name": "gpt-3.5-turbo"
}
query = {
    "messages": [ {
        "role": "system",
        "content": "You are a helpful assistant to help design api structure"
    },
    {
        "role": "user",
        "content": "Hi There!"
    }]
}
response = {
    "format": "json_object"
}
response = zella_ai.chat.completions.create(user, model, query, response)
```
